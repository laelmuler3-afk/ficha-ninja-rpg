# Ficha Ninja RPG — v2.1.1

Aplicativo PWA de ficha para o sistema Shinobi, com catálogo de Jutsus, inventário, batalha, efeitos automáticos e Level Up guiado.

## Destaques da versão 2.1.1

- Level Up com benefícios fixos, Vida e Chakra por clã.
- Migração retroativa dos benefícios de níveis anteriores.
- Personagens existentes mantêm PV e Chakra atuais.
- Proteção adicional para fichas antigas no nível 1.
- Resistência do nível 7 exige 3 escolhas diferentes.
- Rank máximo no catálogo, Ataques por Ação na batalha e bloco de recursos.
- Correção da CA de fichas novas antes do preenchimento da Destreza.
- Cache e arquivos de versão alinhados em 2.1.1.

## Instalação

Publique todos os arquivos mantendo a estrutura das pastas. Em atualizações feitas por etapas, envie `version.json` por último.

## Dados locais

As fichas permanecem no armazenamento local do navegador. Antes de uma atualização importante, use o Gerenciador de Dados para exportar um backup.
